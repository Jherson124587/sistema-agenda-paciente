# APIs Doctoc usadas

- `getOrgInfoAPIV2` (basic, sedes, specialties, users)
- `manageUserInfoAPIV2` (get/update calendarInfo)
- `getDayQuotesAPIV2` (quotes by day, by userId, busy_ranges)
- `getPatientQuoteAPIV2` (quotes by patient)
- `manageQuotesAPIV2` (create, update, cancel)
- `managePatientsAPIV2` (create, search)

## Formatos clave
- dayKey: DD-MM-YYYY
- Fechas ISO con Z (UTC). Se controla TZ en UI con `America/Lima` por defecto.
